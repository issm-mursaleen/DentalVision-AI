# OralVision — Dental AI Platform

Two AI-powered modules, one unified interface.

| Module | What it does | Model |
|---|---|---|
| Oral Screening | Detects oral abnormalities & cancer risk (binary + 3-class) | ResNet18 + Grad-CAM++ |
| Cavity Detection | Detects dental caries in X-rays with bounding boxes | YOLOv8 |

## Which Image Goes Where

| Image type | Module to use |
|---|---|
| Intraoral clinical photo (cheek, tongue, lip, palate) | **Oral Screening** |
| Dental X-ray (bitewing / panoramic) | **Cavity Detection** |

## Retrain from Scratch (Oral Models)

```bash
# 1. Build CSV manifests
venv/Scripts/python.exe ml/dataset_split.py

# 2. Train binary model (Normal vs Abnormal)
cd ml && python train.py

# 3. Train disease classifier (Variation / OPMD / Cancer)
cd ml && python train_disease.py

# 4. Evaluate
cd ml && python test.py
cd ml && python test_disease.py
```
Trained weights are saved to `models/` automatically.

---

## Retrain Cavity Model (YOLOv8)

Open `training/DentalVision_YOLOv8_Training.ipynb` in Google Colab.
After training, copy `runs/detect/train/weights/best.pt` → `models/cavity.pt`.
